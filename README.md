# Hotel Booking Cancellation Analysis using SQL

## Overview
This project explores hotel booking data to understand why customers cancel reservations and what factors influence cancellation behavior.

The analysis was completed using SQL only.

---

## Project Objectives
- Understand hotel booking cancellation trends
- Identify customer behavior patterns
- Analyze how booking lead time affects cancellations
- Compare cancellation behavior across countries and hotel types

---

## Data Cleaning
The dataset was prepared by:
- Removing duplicate records
- Handling missing values
- Standardizing inconsistent text fields
- Preparing the data for analysis

---

## Analysis Performed
- Overall cancellation rate
- Cancellation by country
- Customer type analysis
- Booking lead time analysis
- Hotel type comparison

---

## Key Insights
- Early bookings showed higher cancellation tendencies
- Certain customer groups cancelled more frequently
- Cancellation patterns varied across hotel types
- High cancellation rates can affect hotel revenue and planning

- SQL Server
